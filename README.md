# 日记技能

> 随时记录，凌晨整理成篇。把碎片变成有结构的日记。

---

## 功能简介

这是一个 AI 日记管理技能，专为希望长期记录个人思考、碎碎念、工作复盘的用户设计。

**核心逻辑：白天只管存，凌晨自动整理。碎片永远保留。**

---

## 核心特性

- **📝 碎片记录**：随时发送"日记："开头的消息，原文存入飞书文档，不加工不提炼
- **🌙 凌晨自动整理**：每天凌晨 3:00（北京时间）自动将当日碎片整理成摘要、结论、正文
- **📄 文档结构（倒序）**：最新日期在最前面，每个日期下包含灵感碎片 + 整理后的内容
- **🗂️ 永久保留碎片**：灵感碎片永不删除，整理内容叠加在下方

## 配置说明

### 状态文件

技能会读取 `~/.openclaw/workspace/memory/diary-state.json` 来管理日记状态：

```json
{
  "feishu_doc_token": "",
  "feishu_doc_url": "",
  "fragments": [
    {
      "id": "uuid",
      "timestamp": "2026-04-16T21:32:00+08:00",
      "date": "2026-04-16",
      "time": "21:32",
      "content": "fragment text",
      "feishu_block_ids": [],
      "consolidated": false
    }
  ]
}
```

### 飞书文档

首次使用时，技能会自动在飞书云文档根目录创建一个名为「Wanting 日记」的文档，并保存 `doc_token` 到状态文件。

**文档地址固定不变**，后续所有日记操作都写入同一个文档，不会重建。

## 使用方式

### 触发关键词

发送以下格式的消息即可触发记录：

- `日记：今天有一些想法……`
- `帮我记一下：……`
- `记一下：……`

### 定时任务

技能配置了每天凌晨 3:00（北京时间）的定时整理任务，届时会自动：
1. 读取当日所有灵感碎片
2. 生成摘要、结论、正文
3. 叠加到飞书文档中
4. 发送通知告知整理完成

---

## 自定义提示词

技能内置了专门用于整理日记的提示词，遵循以下原则：

- **灵魂优先**：100% 保留原生叙事的语气和个人印记
- **追随心流**：严格遵守原述的逻辑顺序和情感起伏
- **提纯而非替换**：只去除无意义语气词，不改变表达风格
- **放大个人印记**：允许思考不成熟的部分自然流露

## 设计理念

很多人记录日记的痛点是：**记的时候很随意，回头看的时候发现太乱看不下去**。

这个技能的设计思路是：
1. 记录时完全不用想格式，发什么存什么，降低记录阻力
2. 凌晨自动整理，把碎片串联成有结构的日记
3. 碎片永远保留，整理是对碎片的"叠加"而非"替换"，保留记录的真实感
